## 애플리케이션 테스트

### 전략적 원칙

- 입력 데이터에 대한 유연성

    - 입력 데이터 포맷이나 비즈니스 요구사항 변경 발생은 종종 있는 일이므로 유연하게 대처할 수 있어야함.

- 비즈니스 로직 변경에 대한 유연성

    - 파이프라인 내부의 비즈니스 로직이 바뀔 수 있음.

- 결과의 유연성과 원자성

    - 데이터가 얼마나 자주 갱신되는지, 데이터는 완벽한지 (늦게 들어온 데이터는 없는지), 마지막 순간에 데이터가 변경되지 않았는지
